O que é o projeto?

A ideia de desenvolver este projeto surgiu através dos professor Matheus Garrido, a proposta é uma aplicação web interativa que une tecnologia, criatividade e inteligência artificial. Seu principal objetivo é permitir que qualquer pessoa possa gerar imagens únicas e personalizadas a partir de uma simples descrição em texto. Criado com foco na acessibilidade e inovação, o site oferece uma experiência intuitiva para todos que desejam explorar o potencial da IA de maneira prática e divertida. Dessa forma, colocando em prática o que foi aprendido em sala de aula


Como ele funciona?


O funcionamento é simples: o usuário acessa o site, insere uma descrição textual no campo indicado e, com apenas um clique, a inteligência artificial interpreta essa descrição e gera uma imagem correspondente. Todo o processo acontece online, sem a necessidade de instalações ou conhecimentos técnicos. O sistema combina linguagens de programação modernas com modelos avançados de IA para transformar palavras em imagens de forma rápida e criativa.

Tecnologias usadas

  • HTML

    • HTML (HyperText Markup Language) é a base estrutural da aplicação. Ele define a organização e os elementos do site, como textos, imagens, links, formulários e tabelas. É a espinha dorsal de todas as páginas exibidas.


  • CSS

    • CSS (Cascading Style Sheets) foi usado para dar estilo ao site, definindo cores, fontes, espaçamentos, layouts responsivos e efeitos visuais. Com ele, garantimos uma interface agradável, clara e adaptada para diferentes tamanhos de tela.


  • JavaScript

    • JavaScript adiciona interatividade ao site. Ele é responsável por tornar os elementos dinâmicos e por permitir que os usuários interajam com a inteligência artificial, além de cuidar de validações, manipulação do DOM e funcionalidades em tempo real.


  • Inteligência Artificial

    • A IA é o coração do gerador de imagens. Por meio de modelos avançados de machine learning, ela interpreta o texto fornecido pelo usuário e gera imagens únicas com base na descrição. Essa tecnologia permite experiências criativas e personalizadas que evoluem constantemente com o uso.
